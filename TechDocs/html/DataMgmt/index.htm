GEOS SDK Documentation: Data Management Topics

| GEOS SDK TechDocs

The following documents describe API for certain specialized data-storage mechanisms.

DataStore library
Stores a database in a file. Databases consist of records, each record containing several fields of data.
Cell Library
Stores "cells" of data in a two-dimensional array in a VM file.
Impex Library
Translates GEOS data files into common formats used on PCs.
Contacts Database
The Nokia 9000i Communicator has a database dedicated to storing Contact information such as names, addresses, and phone numbers.
