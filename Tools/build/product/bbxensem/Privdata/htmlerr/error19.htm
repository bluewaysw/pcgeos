Check Internet settings

There is a problem with the Internet dial-up configuration:

  1. Open "Dial-up & Configure."
  2. Ensure an ISP is entered using "Add New ISP" from the ISPs menu.
  3. If a default ISP is not selected, select the ISP you want as the default ISP and click on the "Set as the default ISP" check box.
  4. Once the problem is fixed, you may click "Reload" in the "View" menu to try loading this page again.
