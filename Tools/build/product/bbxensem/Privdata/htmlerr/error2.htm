The web page address can't be loaded

There was a technical error, possibly because too much of your computer's memory is currently in use, or because a file has become corrupted. to solve this, please try these steps (you may want to write these steps down or print this page so you remember them):

1. Close any other programs you have open, then try clicking "Reload" in the "View" menu.

2. If this message appears again, close WebMagick by clicking "File", "Exit WebMagick".

3. Then, restart WebMagick and try viewing the web page again.

4. If this message still appears after that, it's possible a file has become corrupted.
