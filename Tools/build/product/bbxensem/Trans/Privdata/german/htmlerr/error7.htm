Es gibt ein Problem mit der Internetverbindung

Die angeforderte Webseite kann nicht kontaktiert werden, da Ihr Computer nicht mit dem Internet verbunden ist.

Um das Problem zu lösen, führen Sie die folgenden Schritte aus. (Eventuell notieren Sie diese Schritte oder drucken diese Seite aus, um sie später nochvollziehen zu können):

  1. Stellen Sie sicher, dass Ihr Computer mit einem Modem oder Router verbunden ist.
  2. Stellen Sie weiterhin sicher, dass Ihr Modem oder Router mit Ihrem ISP verbunden ist.
  3. Klicken Sie in der Werkzeugleiste auf "Aktualisieren", damit WebMagick erneut eine Verbindung zum Internet herstellt. Alternativ klicken Sie im Menü "Ansicht" auf "Aktualisieren (neu laden)".
  4. Wenn Sie mit dem Internet verbunden sind, aber weiterhin diese Nachricht erhalten, beenden Sie PC/GEOS, schalten den Computer aus, warten einen Moment und schalten dann den Computer wieder ein.
  5. Wenn der Computer neu gestartet ist, überprüfen Sie die Verbindung zu Modem oder Router um sicherzustellen, dass sie noch einwandfrei funktioniert.
  6. Nachdem der Computer gestartet ist, versuchen Sie erneut, mit WebMagick eine Verbindung zum Internet herzustellen.
