Es gibt ein Problem mit dem Modem oder der Telefonleitung

Die angeforderte Webseite kann nicht kontaktiert werden, da Ihr Computer nicht mit einer Telefonleitung verbunden ist oder es ein Problem mit Ihrem Modem oder der Telefonleitung gibt.

Um das Problem zu lösen, versuchen Sie die folgenden Schritte (Sie möchten diese Schritte eventuell notieren oder diese Seite ausdrucken):

  1. Stellen Sie sicher, dass Ihr Computer mit der Telefonleitung verbunden ist.
  2. Stellen Sie sicher, dass Ihr Modem weiterhin mit Ihrem ISP verbunden ist. Versuchen Sie, auf "Neu laden" im Menü "Ansicht" zu klicken, damit der Computer erneut eine Verbindung zum Internet herstellt.
  3. Wenn Sie verbunden sind, aber weiterhin diese Nachricht erhalten, beenden Sie Ensemble, schalten Sie den Computer aus, warten Sie einen Moment, und schalten Sie den Computer dann wieder ein.
  4. Während der Computer neu startet, überprüfen Sie die Telefonleitung mit einem normalen Telefon, um sicherzustellen, dass sie noch einwandfrei funktioniert. Wenn Sie ein Freizeichen hören und eine Nummer wählen können, funktioniert sie korrekt.
  5. Nachdem der Computer gestartet ist, versuchen Sie erneut, mit WebMagick eine Verbindung zum Internet herzustellen.
