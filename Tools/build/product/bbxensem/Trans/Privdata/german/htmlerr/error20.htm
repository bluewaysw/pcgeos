Überprüfen Sie die Internet-Einstellungen bzw. Ihre Telefonleitung!

Ihr Computer kann keine Verbindung zum Internet herstellen.

- Stellen Sie sicher, dass Ihre Internet-Einstellungen richtig sind.

- Stellen Sie sicher, dass die Telefonleitung eingesteckt ist und funktioniert.

Wenn das Problem beseitigt ist, klicken Sie auf "Aktualisieren (neu laden)" im "Ansicht"-Menü.
