Fehler beim Laden der Seite. Stellen Sie sicher, dass JavaScript im Menü "Ansicht" aktiviert ist.
  1. Überprüfen Sie im Menü "Ansicht", ob JavaScript aktiviert ist. ("JavaScript aktivieren" sollte ein Häkchen haben; klicken Sie darauf, wenn dies nicht der Fall ist.)
  2. Wenn JavaScript aktiviert ist, überprüfen Sie die Schreibweise des ersten Teils der Webadresse (der Teil vor "//"), die Sie anzeigen möchten. Es sollte dort http://, https://, ftp://, file://, chat://, mailto:// oder javascript:// stehen.
  3. Wenn Sie sicher sind, dass der erste Teil der Adresse korrekt ist, versuchen Sie möglicherweise, auf eine Art von Internetadresse zuzugreifen, die der Browser WebMagick nicht unterstützt.
