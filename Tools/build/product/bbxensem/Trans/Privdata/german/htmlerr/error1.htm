Fehler beim Laden der Seite. Stellen Sie sicher, dass JavaScript aktiviert ist (im Menü "Ansicht"). Wenn JavaScript aktiviert ist, die Webadresse korrekt eingegeben wurde und diese Nachricht erneut erscheint, unterstützen wir diesen Typ von Webseite möglicherweise nicht.
  1. Bitte klicken Sie im Menü "Ansicht", um zu überprüfen, ob JavaScript aktiviert ist. ("JavaScript aktivieren" sollte ein Häkchen haben; klicken Sie darauf, wenn dies nicht der Fall ist.)
    Wenn JavaScript aktiviert ist, überprüfen Sie bitte die Schreibweise des Anfangs der Webadresse (der Teil vor "//"), die Sie anzeigen möchten. Es sollte http://, https://, ftp://, file://, chat://, mailto:// oder javascript:// sein.
  2. Wenn Sie sicher sind, dass der Anfang der Adresse korrekt ist, versuchen Sie möglicherweise, auf eine Art von Internetadresse zuzugreifen, die der WebMagick-Browser nicht unterstützt.
