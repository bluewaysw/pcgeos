Internet-Einstellungen überprüfen

Es gibt ein Problem mit der Internet-Einwahlkonfiguration:

  1. Öffnen Sie "Einstellungen > Internet-Einwahl" im Express-Menü; die Dialogbox "Internet-Einwahl" wird geöffnet.
  2. Stellen Sie sicher, dass ein Internet Servic Provider (ISP) eingetragen ist. Sollte das nicht der Fall sein, öffnen Sie das Menü "Provider" und klicken auf "Provider hinzufügen"; die Dialogbox "Provider hinzufügen" wird geöffnet. Tragen Sie hier die Daten für einen ISP ein.
  3. Wenn mehrere ISP konfiguriert, aber kein Standard-ISP ausgewählt ist, markieren Sie den gewünschten ISP und klicken auf "Aktivieren".
  4. Wenn das Problem behoben ist, klicken Sie in der Werkzeugleiste auf "Aktualisieren", um diese Seite erneut zu laden. Alternativ klicken Sie im Menü "Ansicht" auf "Aktualisieren (neu laden)".
