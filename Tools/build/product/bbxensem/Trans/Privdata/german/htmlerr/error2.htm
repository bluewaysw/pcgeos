Die Webadresse kann nicht geladen werden

Es gab einen technischen Fehler. Möglicherweise ist zu viel Arbeitsspeicher Ihres Computers aktuell in Gebrauch, oder eine Datei ist beschädigt. Um das Problem zu lösen, führen Sie die folgenden Schritte aus. (Eventuell notieren Sie diese Schritte oder drucken diese Seite aus, damit Sie sie später nachvollziehen können):

  1. Haben Sie weitere Programme gestartet, schließen Sie diese. Klicken Sie dann auf die Schaltfläche "Aktualiseren" in der Werkzeugleiste. Alternativ klicken Sie im Menü "Ansicht" auf "Aktualisieren (neu laden)".
  2. Wenn diese Nachricht erneut erscheint, schließen Sie WebMagick. Klicken Sie dazu im Menü "Datei" auf "Beende WebMagick" oder drücken Sie die Taste F3. Alternativ klicken Sie in der Titelleiste des Programms auf den Schließen-Knopf.
  3. Starten Sie dann WebMagick neu und versuchen Sie, die Webseite erneut zu laden.
  4. Wenn diese Nachricht auch dann weiterhin erscheint ist es möglich, dass eine Datei beschädigt wurde.
