Die Webadresse kann nicht geladen werden

Es gab einen technischen Fehler, möglicherweise, weil zu viel Arbeitsspeicher Ihres Computers aktuell in Gebrauch ist, oder weil eine Datei beschädigt wurde. Um das Problem zu lösen, versuchen Sie bitte die folgenden Schritte (Sie möchten diese Schritte eventuell notieren oder diese Seite ausdrucken, damit Sie sich daran erinnern):

  1. Schließen Sie andere Programme, die Sie geöffnet haben, und versuchen Sie dann, auf "Neu laden" im Menü "Ansicht" zu klicken.
  2. Wenn diese Nachricht erneut erscheint, schließen Sie WebMagick, indem Sie "Datei", "WebMagick beenden" klicken.
  3. Starten Sie dann WebMagick neu und versuchen Sie, die Webseite erneut anzuzeigen.
  4. Wenn diese Nachricht auch danach weiterhin erscheint, ist es möglich, dass eine Datei beschädigt wurde.
